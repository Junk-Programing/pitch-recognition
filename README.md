# 音階当てゲーム

音階を当てる簡単なゲームです。\
下記のリンクで遊べます。\
https://junk-programing.github.io/pitch-recognition/#page-top

## 操作方法

スタートボタンを押すとドレミファソラシのどれかの音がランダムに流れますので、流れた音のボタンを押して当ててください。\
繰り返すボタンを押すと流れた音を聞きなおせます。\
次を聞くボタンを押すと次の問題になります。\
ストップボタンを押すと成績がクリアされます。

## セットアップ～インストール～スタート

リポジトリをクローンします\
\
git clone https://github.com/Junk-Programing/pitch-recognition.git \
\
コマンドにより、pitch-recognitionフォルダ直下で下記のコマンドを使い、必要なモジュールをインストールします。\
\
npm install\
\
下記コマンドで実行できます。\
\
npm start
